這個課程在教什麼？

這是一個為期 5 天的線上課程，由 Google 研究人員與工程師共同打造，從基礎概念深入到可上線的生產級系統，帶你完整學會如何打造強大的 AI agent。課程會聚焦於 vibe coding 工作流程——也就是讓自然語言成為主要的程式介面，並教你如何透過整合工具、API 與多 agent 系統，打造所謂的「10x agent」。

過程中你會建立具備記憶與情境感知能力的「skilled agent」，學習如何評估其品質與安全性，並把專案從原型一路推進到可擴展、可觀測的部署環境。每天的內容都會結合概念深度解析與實作練習、實務引導；課程最後會以一個 capstone 專案作為總結，讓你把學到的東西真正落地。

課程怎麼進行？

活動期間，每位學員每天都會收到以下內容：

  • 📚 每日作業：包含白皮書、配套 podcast 與配套 codelab，可依自己的節奏完成。
  • 💬 Discord 討論：Kaggle Discord 伺服器會開設專屬頻道，提供活動期間的深度討論空間，方便提出問題、釐清疑惑，並與其他學員及 Google 專家交流。
  • 🎥 每日直播與 AMA：每天會在 Kaggle 的 YouTube 頻道進行直播，由作者與課程貢獻者深入講解當日主題並回答問題，過程中也會有一些有趣的互動環節。
  • 💻 Capstone 專案：透過真實世界的專案提升技能並充實作品集。課程結束後可選擇參與評選，角逐 Kaggle 證書、徽章、周邊商品，以及在 Kaggle 與 Google 社群媒體上的曝光機會（此項目為選填）。

每日課程內容

  • Day 1：Agent 與 Vibe Coding 入門 — 從 AI 聊天機器人、文字補全進階到自主 agent，掌握以自然語言作為主要程式介面的 vibe coding 工作流程。
  • Day 2：Agent 工具與互通性 — 透過整合外部 API、程式碼執行與 agent 對 agent 通訊，將 agent 的能力擴展到近乎無限。
  • Day 3：Agent Skills — 打造具備長期記憶與狀態管理的個人化 agent，掌握長情境與最佳 token 使用策略，並將技能整合進 agentic framework。
  • Day 4：Vibe Coding Agent 安全性與評估 — 透過嚴謹的測試、護欄機制與品質評估打造可靠的 agent，並針對新型威脅向量進行安全防護。
  • Day 5：Vibe Coding 時代下的規格驅動正式環境開發 — 將本地 agent 升級為受治理、可擴展、可觀測的正式環境艦隊，掌握雲端部署、除錯與可觀測性。

這次課程的新亮點

  • Vibe Coding 工作流程：深入探討自然語言如何成為打造自主 agent 的全新「程式語言」。
  • 全新內容與講師陣容：邀請新領域專家，帶你了解如何打造「10x agent」「skilled agent」「quality agent」。
  • 從 Vibe 到 Live：全新聚焦於如何把本地 agent 升級為受治理、可擴展、可觀測的企業級艦隊，並以全新的 Kaggle capstone 專案作為總結。

常見問題

有先修要求嗎？適合什麼樣的人參加？
本課程適合任何對打造與理解 AI agent 有興趣的人。雖然沒有正式的先修門檻，但具備以下基礎會更有幫助：
  • Python 程式設計：課程包含實作練習，建議具備 Python 基礎。
  • 基礎 AI 概念：若還不熟悉，建議先從「5-Day AI Agent Intensive Course with Google」入門。
  • Google AI Studio：熟悉 Google AI Studio 有助於實驗與部署 AI 模型（API 與 AI Studio 的可用地區請參考官方文件）。
每天大概要花多少時間？
每日作業約需 1–2 小時完成，每日直播約 45–60 分鐘。
課程內容怎麼發布？
活動週期間，每天的課程內容會透過你註冊的 email、本頁的討論區，以及 Kaggle Discord 發布。
怎麼加入 Kaggle 的 Discord？
註冊一個 Discord 帳號，並加入 Kaggle Discord 伺服器即可。
Capstone 專案是什麼？
透過實作真實世界的 agent 專案來提升技能、充實作品集。提交內容須包含一篇 Kaggle 文章，記錄你打造的 agent，包括說明影片、建構理念簡述，以及程式碼連結。
Capstone 專案的時程？
專案公開上線：2026 年 6 月 19 日（週五）當天結束。提交截止：2026 年 7 月 6 日（週一）23:59 PT。
課程會頒發 Kaggle 證書、獎牌或積分嗎？
參與 capstone 專案即可在 Kaggle 個人檔案上獲得徽章與證書。優勝者另可獲得 Kaggle 限定周邊，並在 Kaggle 與 Google 社群媒體上獲得表揚。
直播會錄影嗎？
會。無法即時觀看直播的學員，事後將收到完整錄影檔。
課程費用是多少？
完全免費，只需要一個 Kaggle 帳號與一個 Google AI Studio 帳號。注意：設定 Kaggle 帳號時請務必完成手機號碼驗證，以便活動週能順利參與。
設定 Kaggle 帳號一定要驗證手機號碼嗎？
是的，這是必填項目。驗證步驟如下：
  1. 點選 Kaggle 個人頁面右上角的頭像，選擇「Settings」。
  2. 在「Phone Verification」項目下，點選「Verify Phone」。
  3. 依照畫面指示完成驗證。
輸入電話號碼時請記得加上國碼，國碼前不要加任何數字或符號。
哪些人可以使用課程資源？
直播與錄影內容對所有人開放，但部分工具（如 Gemini API、Kaggle notebook 等）以下對象將無法使用：
  • 受制裁國家居民
  • 未滿 18 歲者
